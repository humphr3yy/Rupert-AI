Rupert Discord Bot

Your psychedelic AI voice assistant for Discord voice channels

Voice Transcription

Transcribes everything said in your Discord voice channel in real-time.

AI Chat Assistant

Interact with Rupert AI by just saying "Rupert" in your conversations.

Text-to-Speech

Responses are played back through natural-sounding Piper TTS voice.

Setup Instructions

  1. Create a Discord bot token in the Discord Developer Portal
  2. Add the bot to your Discord server with voice permissions
  3. Set up Ollama on your local machine (see below)
  4. Configure your environment variables
  5. Run the bot using the commands below

Prerequisites

  • Python 3.8+
  • Discord bot token
  • Ollama - Local large language model
  • Piper TTS - Text-to-speech engine

Environment Variables

DISCORD_TOKEN - Your Discord bot token
OLLAMA_HOST - Hostname for Ollama (default: localhost)
OLLAMA_PORT - Port for Ollama API (default: 11434)
OLLAMA_MODEL - Ollama model name (default: llama2)
RUN_BOT_DIRECTLY - Set to "true" to run the bot directly

Bot Commands

  • !join Join your voice channel
  • !leave Leave the voice channel
  • Hey Rupert, [your question] Talk to Rupert AI
